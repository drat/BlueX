BlueX

BlueX is "DDoS-like" tool which will try to ping any IP / Web address, with numer of connections you entered. This is simlpe Python script, so if you´re using Windows, you will need Python 2.7. There are compiled versions for Windows, if you don´t want to install Python 2.7, and .py script for Linux. This should also work on Mac OS X 10.4+

Useful links:
